Auto Home

Open Source Home Automation

View project onGitHub

This project aims to remotely control home via internet with the help of Arduino.

They are supplied complete electronic design and source code where anyone with minimal knowledge of electronics can implement home automation on the cheap and safe.

So is open source!

Besides being open another goal is to be as educational as possible, trying to guide the user clearly at all stages.
